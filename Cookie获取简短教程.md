# 微博 Cookie 获取简短教程

这份教程只讲最短路径，按步骤做基本就能成功。

## 方式一（推荐，最省事）
1. 运行 `.\.venv\Scripts\python app.py` 打开本工具。
2. 点击 `打开调试 Edge`。
3. 在新打开的 Edge 窗口登录 `https://weibo.com`（第一次需要登录，之后会保存在本项目的 `.edge_cdp_profile/`）。
4. 回到工具页面，点击 `自动获取 Cookie`。
5. 成功后会自动填入 Cookie，直接开始抓取即可。

如果失败：
- 先确认调试 Edge 窗口里微博确实是登录状态。
- 不要关闭调试 Edge 窗口，保持它开着再点 `自动获取 Cookie`。
- 仍失败就用“方式二”。

## 方式二（手动复制，最稳）
1. 在浏览器打开并登录微博任意页面（例如个人主页）。
2. 按 `F12` 打开开发者工具，切到 `网络 / Network`。
3. 刷新页面，点任意一个 `weibo.com/ajax/...` 请求。
4. 在 `请求标头 / Headers` 里找到 `Cookie`（或用“Copy as cURL”后提取 `-b` 后面的整串）。
5. 复制整段 Cookie，回到本工具粘贴到 `微博 Cookie` 输入框，再点击 `识别粘贴内容`。

## 复制时的注意事项
- 要复制完整一整串，不要只复制 `SUB` 或 `XSRF-TOKEN` 其中一个。
- 前后不要多空格或换行。
- Cookie 有时效，失效后重新获取一次即可。

## 常见报错与处理
- `访客验证/未登录`：Cookie 失效或不完整，重新登录后重新复制。
- `接口返回异常`：先确认微博网页能正常访问，再重试。
- `自动获取失败`：改用手动复制方式。

## 安全提醒
- Cookie 等同于登录态，请勿发给他人。
- 用完建议在微博“退出登录”或清理浏览器登录态。
